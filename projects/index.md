## EN

This section presents what I have built during my academic path.

While GitHub repositories contain the source code, this page provides context: objectives, architectural decisions, technical challenges and security considerations behind each project.

Each project is described through:

1. **Context** – why the project was developed;  
2. **Objective** – what problem it aims to solve;  
3. **Architecture / Technical Design** – how it is structured;  
4. **Key Technical Challenges** – the main engineering difficulties encountered;  
5. **Security Perspective** – security-related implications or lessons learned;  
6. **Repository** – link to the implementation.

## IT

Questa sezione presenta cosa ho sviluppato e studiato durante il mio percorso accademico.

Mentre le repository su GitHub contengono il codice, qui viene fornito il contesto: obiettivi, scelte architetturali, difficoltà tecniche e considerazioni di sicurezza dietro ogni progetto.

6. **Repository** – link all’implementazione.
